# vscode-plugin-ncchr-lang README

用友NCCHR项目多语辅助工具

## Features

选中需要替换的汉字，执行后在多语文件中添加相应对象，选中汉字替换为该对象key值

## Usage

* `Ctrl+Shift+P`弹出对话框，在对话框中输入ncchr-lang.replace
* 输入多语前缀，一般为json、language、this.state.json等
* 选择多语文件，一般一个项目下会有多个多语文件

**Enjoy!**
